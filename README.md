# Event Scheduler System

This is a Python-based Event Scheduler System that manages event scheduling and detects overlapping time conflicts between events.

## Technologies Used
- Python

## Features
- Add events with start and end times
- Detect scheduling conflicts
- Prevent overlapping events
- Store scheduled events
- Menu-driven workflow

## Concepts Used
- Lists
- Loops
- Conditional Statements
- User Input
- Time Conflict Detection Logic

## Project Type
Python Console Application
